Working with colors and animations

About the assignment

To solve this assignment, I did it in a three-part approach. First, I used Huemint to create a color palette and experimented with it using a color tester assignment. I validated the contrast using Adobe's Contrast Analyzer and then applied the palette to the website. I created CSS variables for all the colors and applied them, experimenting with different nuances to make the design look better.


Next, I created a dark theme using a method with CSS Variables and JavaScript.


Finally, I added animations to my design. This was the last part of the assignment. Overall, the assignment helped me improve my skills in working with colors and animations. I learned how to create an effective color palette using Huemint, how to experiment with different nuances, and how to validate contrast using Adobe's Contrast Analyzer. I also learned how to create a dark theme using CSS variables and local storage, as well as how to add animations to my designs.
