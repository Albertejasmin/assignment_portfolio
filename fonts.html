Fonts and UI components

About the assignment

The assignment was to create a mockup landing page design by using fonts and gray scales to begin with. We had to implement a font hierarchy and design UI cards.


To create a font hierarchy, I started by verifying the hierarchy was working using Type Scale. Then, I applied the hierarchy to the website using font sizes set in rem as I learned in class. I made sure to work with not just font-size, but also line-height, margins, and paddings, as they are also a significant part of setting type. To find inspiration, I looked at landing pages on Land-book's gallery and found good exsamples of how to use fonts to create a nice hierarchy.


In the second part of the assignment, I improved the UI components by following the guidelines for designing UI components and with the CSS techniques that I know. Lastly, I apply a simple color pallet. possible.
