Responsive design

About the assignment

To solve the Responsive Design assignment, I first looked at the HTML in the browser and read through the HTML in VS Code to recognize which elements were components and which should be layouts. I then used CSS Flex and Grid methods, as well as min width/height and media queries to layout the page.


I referred to the HTML and CSS I had used throughout the course for guidance. Using Flex and Grid allowed me to create a responsive layout that adapts to different screen sizes and orientations.


By breaking down the page into components and layouts, and using CSS Flex and Grid, I was able to create a responsive design that meets the requirements of the assignment.
